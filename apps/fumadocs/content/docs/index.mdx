## Introduction

Cobalt is a financial platform that connects your accounts, transactions, brokerage data, and market research in one place.

## MCP Server

Connect AI agents (Claude, Cursor, VS Code, Zed, Codex, Gemini, Amp, OpenCode) to your financial data via the [Model Context Protocol](/docs/mcp). Agents call a single `cobalt_execute_code` tool to run JavaScript against the `cobalt.*` SDK in a sandboxed V8 isolate — accounts, transactions, brokerage positions, tags, and market research, all auto-scoped to the authenticated user.
